# 스프라이트 추출 도구 — 단순 모드 구현 과업

> 가장자리 흰색 기준 BFS 탐색 + 경계면 알파 처리

---

## 크게 쪼개기

```
단순 모드 스프라이트 추출
  → 이미지 로드 및 픽셀 데이터 접근
  → 흰색 판별 기준 정의
  → 가장자리 기반 BFS 탐색으로 배경 영역 마킹
  → 마킹된 배경 영역 삭제 (알파 0)
  → 경계면 감지
  → 경계면 알파 처리 (흰색 비례 투명도)
  → 결과 이미지 출력 (PNG)
```

---

## 기능 단위로 쪼개기 + 완료 조건

### 1. 이미지 로드 및 픽셀 데이터 접근

```
□ 이미지 파일을 RGBA 배열로 로드
  → 테스트: 10x10 흰색 PNG 로드 시 모든 픽셀이 (255,255,255,255)이면 완료
```

### 2. 흰색 판별 함수

```
□ is_white(pixel, threshold) 함수 구현
  → (255,255,255) → True
  → (250,251,253) threshold=10 → True
  → (200,200,200) threshold=10 → False
  → 세 케이스 모두 통과하면 완료
```

### 3. 가장자리 기반 BFS 배경 탐색

```
□ 이미지 가장자리(상하좌우 테두리)의 흰색 픽셀을 시작점으로 수집
  → 테스트: 10x10 이미지, 테두리가 모두 흰색일 때 시작점 36개(테두리 픽셀 수)면 완료

□ BFS로 시작점에서 인접 흰색 픽셀까지 확장하여 배경 영역 마킹
  → 테스트: 중앙에 5x5 검정 사각형이 있는 20x20 흰색 이미지
    - 배경 마킹 영역 = 전체 - 검정 영역
    - 검정 사각형 내부는 마킹되지 않으면 완료
```

### 4. 배경 영역 삭제

```
□ 마킹된 배경 픽셀의 알파를 0으로 설정
  → 테스트: 3번의 결과에서 마킹된 영역의 알파가 모두 0,
    마킹되지 않은 영역의 알파가 모두 255이면 완료
```

### 5. 경계면 감지

```
□ 오브젝트(비배경) 픽셀 중 배경과 인접한 픽셀을 경계면으로 식별
  → 테스트: 중앙에 5x5 검정 사각형이 있는 20x20 흰색 이미지
    - 경계면 = 검정 사각형의 외곽 테두리 픽셀(16개)이면 완료

□ 경계면에서 안쪽으로 N픽셀(설정 가능) 범위까지 확장
  → 테스트: depth=2 설정 시 경계면 + 안쪽 1픽셀 레이어까지 포함되면 완료
```

### 6. 경계면 알파 처리

```
□ 경계면 픽셀의 밝기(흰색 근접도)를 알파에 반비례 매핑
  → 공식: alpha = 255 - (흰색 근접도 * 255)
  → 테스트:
    - 픽셀 (255,255,255) → alpha ≈ 0 (거의 투명)
    - 픽셀 (128,128,128) → alpha ≈ 128 (반투명)
    - 픽셀 (0,0,0) → alpha = 255 (불투명)
    - 세 케이스 통과하면 완료
```

### 7. 결과 출력

```
□ 처리된 RGBA 데이터를 PNG로 저장
  → 테스트: 출력 파일이 유효한 PNG이고, 알파 채널이 포함되어 있으면 완료
```

## 1차 사이클 (오늘의 투두)

```
□ 이미지 RGBA 로드 함수
□ is_white 판별 함수
□ 가장자리 시작점 수집 함수
□ BFS 배경 영역 마킹 함수
□ 배경 영역 알파 0 처리 함수
□ PNG 출력 함수
→ 통합 테스트: 흰 배경 위 검정 원 이미지 입력 → 배경만 투명하게 된 PNG 출력
```

## 기술 선택

- **언어:** Python (PIL/Pillow 기반, 프로토타입에 적합)
- **포맷:** 입력은 PNG/JPEG, 출력은 반드시 PNG (알파 채널 필요)
- **테스트:** pytest + 테스트용 이미지 자동 생성 (Pillow로 직접 그림)
