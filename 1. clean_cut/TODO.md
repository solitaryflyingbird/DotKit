# clean_cut UI 개선 목록

## 1. 도구 커서 프리뷰

현재 펜/지우개/올가미 사용 시 마우스 위치에 아무 표시가 없어 감으로 작업해야 한다.

- 펜/지우개: 현재 브러시 크기에 맞는 원형 커서를 마우스 위치에 표시
- 올가미: 십자 또는 포인트 커서 표시
- 줌 레벨에 따라 커서 크기 연동

## 2. 작업대 줌 슬라이더

현재 휠로만 줌 조절이 가능해 불편하다.

- 작업대 하단에 줌 레벨 조절 바(슬라이더) 추가
- 브러시 크기 슬라이더와 유사한 형태
- 현재 줌 배율 수치 표시

## 3. 투명 배경색 선택

PNG 투명 영역의 배경색을 선택할 수 있어야 한다.

- 작업대와 결과 패널에 동시 적용
- 기본값: 체커보드 (현재 동작)
- 단색 배경 옵션 (흰색, 검정, 커스텀 색 등)
