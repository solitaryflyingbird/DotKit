# clean_cut — 스프라이트 선따기 도구

흰색 배경 위의 2D 게임 스프라이트를 자동으로 오려내는 도구입니다.

## 1단계: 배경 제거 + 경계면 알파 처리

### 동작 원리

1. **BFS 배경 탐색** — 이미지 가장자리(테두리)에서 출발하여 흰색 픽셀을 BFS로 탐색, 배경 영역으로 마킹
2. **배경 삭제** — 마킹된 배경 픽셀의 알파를 0으로 설정
3. **경계면 알파 처리** — 오브젝트와 배경이 맞닿은 경계면 픽셀에 밝기(흰색 근접도) 비례 투명도 적용. 안티앨리어싱 잔여물을 자연스럽게 페더링

### 핵심 원칙

- 경계면 알파는 **밝기 기반 일괄 적용**. 채도 조건을 넣으면 안 됨
- 경계면 픽셀은 "오브젝트 색 + 흰 배경"이 섞인 결과물이므로, 밝기로 알파를 먹이면 원래 색이 복원됨

### 사용법

```bash
python clean_cut.py <입력 이미지> <출력 PSD> [threshold]
```

- `threshold`: 흰색 판별 허용 오차 (기본값 10, RGB 각 채널 기준)
- 출력 PSD에는 `cut`(선따기 결과) + `bg`(확인용 보색 배경) 레이어 포함

### 예시

```bash
python clean_cut.py sprite.png sprite_cut.psd
python clean_cut.py sprite.jpg sprite_cut.psd 15
```
