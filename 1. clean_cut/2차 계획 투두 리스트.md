# 마스크 모드 구현 과업

> 마스크 파일 기반 영역 지정 → 해당 영역 내 배경 제거 + 경계면 알파 처리

---

## 개요

단순 모드(1차)는 이미지 가장자리에서 BFS로 배경을 탐색했다.
마스크 모드는 별도의 마스크 파일이 "여기가 배경이다"라고 지정한 영역을 처리한다.

- 마스크 파일은 원본과 동일한 가로×세로
- 마스크 형식: 불투명 색상이 있는 픽셀 = 처리 대상 영역
  - 검정, 빨강, 파랑 등 색상 무관
  - 투명 배경 PNG에 색칠된 형태도 가능
  - 판별 기준: 알파 > 0 이고 완전한 흰색이 아닌 픽셀
- 마스크 영역 내에서 채도·명도가 동시에 낮은 픽셀(흰색/회백색)을 배경으로 판정하여 제거
- BFS/DFS 탐색 없이 마스크 영역 전체를 일괄 처리

---

## 크게 쪼개기

```
마스크 모드 스프라이트 추출
  → 마스크 파일 로드 및 유효성 검증
  → 마스크 영역 추출 (어떤 픽셀이 처리 대상인지)
  → 마스크 영역 내 배경 판별 (채도·명도 기준)
  → 배경 픽셀 삭제 (알파 0)
  → 경계면 감지 및 알파 처리
  → 결과 이미지 출력 (PNG)
```

---

## 기능 단위로 쪼개기 + 완료 조건

### 1. 마스크 로드 및 유효성 검증

```
□ 마스크 파일을 RGBA 배열로 로드
  → 테스트: 유효한 마스크 PNG 로드 성공

□ 원본과 마스크의 가로×세로가 동일한지 검증
  → 테스트: 크기가 다르면 에러 발생, 같으면 통과
```

### 2. 마스크 영역 추출

```
□ 마스크에서 "처리 대상" 픽셀을 판별하는 함수 구현
  → 조건: 알파 > 0 이고 완전한 흰색(255,255,255)이 아닌 픽셀
  → 테스트:
    - 검정(0,0,0,255) → 대상 ✓
    - 빨강(255,0,0,255) → 대상 ✓
    - 파랑(0,0,255,255) → 대상 ✓
    - 흰색(255,255,255,255) → 대상 ✗
    - 투명(0,0,0,0) → 대상 ✗
    - 반투명 색상(128,0,0,128) → 대상 ✓
    - 모든 케이스 통과하면 완료
```

### 3. 마스크 영역 내 배경 판별

```
□ 원본 픽셀의 채도(Saturation)와 명도(Value)를 계산하는 함수 구현
  → HSV 변환 기반
  → 테스트:
    - (255,255,255) → S=0, V=1.0 (흰색)
    - (200,200,200) → S=0, V≈0.78 (회색)
    - (255,0,0) → S=1.0, V=1.0 (순수 빨강)
    - (0,0,0) → S=0, V=0 (검정)

□ 마스크 영역 내에서 채도·명도 기준으로 배경 픽셀을 판별
  → 조건: 채도 ≤ threshold_s AND 명도 ≥ threshold_v → 배경
  → (채도 낮고 명도 높은 = 흰색/회백색 계열)
  → 테스트:
    - 흰색(255,255,255) → 배경 ✓
    - 밝은 회색(230,230,230) → 배경 ✓
    - 진한 빨강(200,30,30) → 배경 ✗
    - 어두운 회색(50,50,50) → 배경 ✗ (명도 낮음)
    - 모든 케이스 통과하면 완료
```

### 4. 배경 영역 삭제

```
□ 마스크 영역 내 배경으로 판별된 픽셀의 알파를 0으로 설정
  → 테스트: 마스크 영역 밖의 픽셀은 변경 없음
  → 테스트: 마스크 영역 내 배경 픽셀만 알파 0
```

### 5. 경계면 감지 및 알파 처리

```
□ 마스크 영역 내에서 삭제된 배경과 인접한 오브젝트 픽셀을 경계면으로 식별
  → 1차에서 구현한 find_boundary / expand_boundary / apply_boundary_alpha 재활용

□ 경계면 픽셀에 밝기 기반 알파 매핑 적용
  → 테스트: 1차와 동일한 알파 매핑 규칙 적용 확인
```

### 6. 결과 출력

```
□ 처리된 RGBA 데이터를 PNG로 저장
  → 테스트: 출력 파일이 유효한 PNG이고 알파 채널 포함
```

---

## 1차 사이클 (투두)

> 마스크 영역 추출 + 채도·명도 기반 배경 판별 + 삭제

```
□ 마스크 로드 및 크기 검증 함수
□ 마스크 영역 추출 함수
□ 채도·명도 계산 함수
□ 마스크 영역 내 배경 판별 함수
□ 배경 영역 알파 0 처리 함수
□ PNG 출력
→ 통합 테스트: 마스크가 지정한 영역의 흰색 배경만 투명 처리
```

## 2차 사이클

> 경계면 알파 처리 추가 (1차 코드 재활용)

```
□ 마스크 영역 내 경계면 감지
□ 경계면 알파 처리 적용
→ 통합 테스트: 마스크 영역 경계가 부드럽게 페더링된 PNG 출력
```

---

## 기술 메모

- 채도·명도 판별: RGB → HSV 변환 (colorsys 또는 직접 계산)
- 마스크 판별은 색상(Hue) 무시 — 색상이 뭐든 "칠해져 있으면" 대상
- 1차에서 만든 경계면 처리 함수들은 bg_mask만 받으면 되므로 그대로 재활용 가능
