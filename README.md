## Tools

### 1. clean_cut

Background removal with alpha-blended edge processing for pixel art sprites.

*스프라이트 선따기 도구. 흰색 배경을 제거하고, 경계면의 픽셀을 알파 블렌딩하여 깔끔하게 오려냅니다.*
